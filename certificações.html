Certificado Profissional de Análise de Dados do Google

Coursera

O curso "Certificação Profissional de Análise de Dados do Google" oferecido na Coursera é uma qualificação reconhecida que abrange habilidades essenciais de análise de dados usando ferramentas e tecnologias do Google.

Ele aborda tópicos como coleta, limpeza e visualização de dados, uso do Google Cloud Platform e BigQuery, além de técnicas avançadas de análise. Os alunos aprendem a criar insights acionáveis a partir de dados brutos, a fim de impulsionar a tomada de decisões e a inovação.

O programa inclui mais de 180 horas de instrução e centenas de avaliações baseadas na prática, que o ajudarão a simular cenários reais de análise de dados que são essenciais para o sucesso no local de trabalho. O conteúdo é altamente interativo e desenvolvido exclusivamente por funcionários do Google com décadas de experiência em análise de dados. Por meio de uma combinação de vídeos, avaliações e laboratórios práticos, você conhecerá ferramentas e plataformas de análise e as principais habilidades analíticas necessárias para um trabalho inicial.

Machine Learning e Data Science com Python

Udemy

Este curso oferece uma base sólida sobre os principais algoritmos de Machine Learning, incluindo a utilização de bibliotecas como numpy, sklearn e pandas. Você aprenderá sobre classificação, regressão, regras de associação, agrupamento, pré-processamento de dados, redução de dimensionalidade e avaliação de algoritmos usando estatística.

Além disso, o curso aborda a detecção de outliers, a implementação de classificadores para previsões financeiras e salariais, e a relação entre estatística e Machine Learning. Você também terá a oportunidade de implementar algoritmos de regressão, o algoritmo Apriori, técnicas de agrupamento e aprendizagem por reforço.

Por fim, o curso ensina a implementar um classificador de sentimentos em textos, técnicas de Visão Computacional, seleção de atributos, tratamento de bases de dados desbalanceadas e previsões futuras com ARIMA e Facebook Prophet.

Machine Learning e Data Science com R

Udemy

Este curso oferece uma base teórica sólida sobre os principais algoritmos de Machine Learning, utilizando a linguagem R, uma linguagem de programação amplamente utilizada em ciência de dados e aprendizado de máquina. Você terá a oportunidade de aprender sobre uma variedade de algoritmos, incluindo aqueles usados para classificação, regressão, regras de associação e agrupamento. Além disso, o curso ensina como realizar o pré-processamento em bases de dados, uma etapa crucial na preparação de dados para análise.

O curso também aborda técnicas avançadas para redução de dimensionalidade, como Análise de Componentes Principais (PCA), Kernel PCA e Análise Discriminante Linear (LDA). Essas técnicas são essenciais para lidar com conjuntos de dados de alta dimensão. Além disso, você aprenderá a avaliar os algoritmos de Machine Learning usando estatística não paramétrica, um conjunto de métodos estatísticos que não assumem uma distribuição específica dos dados. Por fim, o curso ensina como detectar outliers em bases de dados, o que é fundamental para garantir a qualidade dos dados e a precisão dos modelos de aprendizado de máquina

Microsoft Power BI para Data Science

Data Science Academy

O curso "Microsoft Power BI para Data Science" ministrado pela DSA proporciona uma abordagem abrangente e aprofundada sobre o uso da IDE. O conteúdo engloba desde a instalação inicial, passando pela criação de relatórios e dashboards, até a publicação e compartilhamento de relatórios, bem como a exploração de recursos avançados, como DAX, R e Python.

Além disso, o programa inclui instruções detalhadas sobre os processos de ETL utilizando o Power Query, orientações sobre o design e layout de relatórios mais eficientes e o emprego de técnicas de storytelling para criar relatórios impactantes e de fácil compreensão.

Os participantes aprenderão a realizar a limpeza, transformação e modelagem de dados provenientes de diversas fontes, incluindo planilhas, arquivos de texto, bancos de dados relacionais, além de arquivos CSV e TXT. O curso direciona os alunos para as melhores práticas de análise de dados, mantendo-os atualizados com as exigências do mercado.

Python: Fundamentos para Análise de Dados

Data Science Academy

O curso “Python: Fundamentos para Análise de Dados” da Data Science Academy é um programa abrangente que começa com o básico da programação em Python, incluindo a criação de variáveis e estruturas de repetição, e avança até conceitos intermediários como a criação de funções e classes. O curso também aborda tópicos avançados, como a criação de gráficos com o Matplotlib, a utilização de bibliotecas como NumPy e Pandas, e a realização de análises estatísticas com o SciPy.

Além disso, o curso se aprofunda na aplicação do Python para análise de dados e ciência de dados. Você aprenderá a realizar tarefas como tratamento de valores ausentes, limpeza e processamento de dados, análise estatística descritiva, binarização e codificação de variáveis categóricas, e engenharia de atributos. O curso também oferece uma experiência prática com exemplos, quizzes, exercícios e projetos, permitindo que você aplique seus conhecimentos em projetos reais de análise de dados e ciência de dados.
