Analise de Dados Descritiva
Durante os 9 meses, trabalhei na análise de dados descritivos, que envolveu a interpretação de dados históricos para identificar tendências, padrões e insights. Isso incluiu a geração de estatísticas descritivas e a visualização de dados por meio de gráficos e tabelas.
Analise de Dados Preditiva
Também me envolvi na análise preditiva, utilizando técnicas estatísticas e algoritmos de machine learning para criar modelos que poderiam prever futuros comportamentos e tendências por meio de algoritos classificadores.
Estatistica com R
Ganhei experiência com a linguagem R, outra ferramenta poderosa para análise de dados. Usei R para realizar análises estatísticas intermediárias e visualização de dados com bibliotecas como ggplot2, dplyr e tidyr, ampliando ainda mais meu conjunto de habilidades em análise de dados.
Analise com Python
Utilizei Python, uma linguagem de programação popular na análise de dados, para manipular dados, realizar análises estatísticas e criar modelos de machine learning. Ele/ela se familiarizou com bibliotecas Python como pandas, numpy e scikit-learn.
SQL
Adquiri habilidades em SQL, a linguagem padrão para interagir com bancos de dados. Utilizando consultas SELECT, INNER JOIN entre outras, demonstrando assim a capacidade de efetuar consultas direcionadas em bancos de dados, extrair informações relevantes e realizar análises detalhadas. Além disso, empregou funções SQL para otimizar a manipulação e a transformação dos dados.
Power BI
Por fim, empreguei o Microsoft Power BI para desenvolver relatórios dinâmicos e painéis interativos, incluindo a criação de métricas e indicadores-chave para inteligência de negócios. Nessa capacidade, utilizei a ferramenta para a visualização de conjuntos de dados, facilitando o compartilhamento de insights e contribuindo para a orientação das decisões com base em dados fundamentais.
