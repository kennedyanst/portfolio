Projetos de Sites em Front-end

➣ Projeto Cordel: Durante o desenvolvimento do Projeto Cordel, adquiri competências fundamentais no campo de criação de websites com efeito "parallax". Com a conclusão do curso online de HTML5 e CSS3, fornecido pela plataforma Curso em Vídeo, fui capaz de conceber uma experiência moderna e amplamente apreciada em um site, demonstrando habilidades profissionais e eficazes.

➣ Projeto Login: O Projeto Login foi inspirado em tendências atuais de design de tela de login, amplamente difundidas na comunidade de desenvolvedores do TikTok. Adotei uma abordagem estilística baseada em elementos neon e cores futuristas, empregando imagens geradas com o auxílio de inteligência artificial pelo Canva.

➣ Projeto Parallax: No Projeto Parallax, apliquei a técnica de parallax em conjunto com JavaScript e ScrollReveal. Esta abordagem ofereceu uma estilização distinta aos títulos à medida que o usuário rolava a página. A técnica, notoriamente popular e elegante, acrescentou um toque de sofisticação a páginas de estilo parallax.

➣ Projeto Certificações: A criação deste portfólio resultou na aquisição de diversos conhecimentos que enriqueceram meu repertório no desenvolvimento Front-end. Além de incorporar efeitos "neon" e reflexivos no menu, integrei habilidades em parallax e na concepção de cartões para criar uma página de aspecto moderno e elegante.

➣ Projeto Experiência: Na concepção desta página em particular, minha inspiração veio de um projeto que encontrei no YouTube. Para criar um elemento distinto, desenvolvi imagens com o auxílio da inteligência artificial, representando minhas experiências. Estas imagens foram organizadas lado a lado, com a imagem subsequente ocupando o plano de fundo, criando um efeito de transição notável.

Análise de Dados com Python

➣ Desenvolvimento de Modelos Preditivos para Créditos Bancários: Neste projeto, utilizei uma base de dados amplamente reconhecida na comunidade de machine learning para prever o pagamento de empréstimos por parte dos clientes. Realizei um rigoroso pré-processamento dos dados e desenvolvi modelos classificadores empregando algoritmos de alto desempenho, incluindo Árvore de Decisão, Random Forest, KNN, SVM e Redes Neurais. Utilizei bibliotecas de renome, como Pandas, Numpy, Seaborn, Matplotlib, Yellowbrick, Pickle e Scikit-Learn, para criar e avaliar esses modelos.

➣ Detecção e Tratamento de Outliers para Aumentar a Precisão: Reconhecendo a importância da detecção de outliers na construção de modelos de machine learning mais precisos, utilizei técnicas avançadas, como BoxPlot, e empreguei o detector KNN da biblioteca PyOD. Identifiquei e tratei outliers na base de dados, melhorando assim a precisão e confiabilidade dos modelos gerados.

➣ Otimização dos Modelos por Meio de GridSearch e Análise Estatística: Aprimorei a performance dos algoritmos de modelo por meio da função GridSearchCV do Scikit-Learn, resultando em ganhos significativos de precisão, variando de 0,10% a 0,55%. Além disso, conduzi uma análise estatística abrangente, incluindo validação cruzada, teste de SHAPIRO, análise da NORMAL, teste ANOVA e Tukey para garantir a robustez e confiabilidade dos modelos.

➣ Combinação Inteligente de Modelos para Maior Confiança nos Resultados: Implementei uma estratégia de combinação dos melhores modelos classificadores, utilizando uma estrutura lógica simples baseada em regras condicionais. Essa abordagem resultou em uma maior confiabilidade dos resultados, com base na maioria das previsões dos modelos individuais.

➣ Visualização Avançada de Dados com Gráficos Impactantes: Tenho conhecimento na criação de uma variedade de gráficos e visualizações de dados impressionantes, abrangendo desde os gráficos básicos até gráficos 3D. Utilizando as principais bibliotecas Python para produzir representações visuais que enriqueceram a compreensão dos dados e insights obtidos.
