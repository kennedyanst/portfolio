Um exemplo de projeto que desenvolvi para a área da estatística, é o SIADE (Sistema de Análise de Dados Estatísticos).

É uma ferramenta que facilita a análise de dados estatísticos, permitindo que usuários realizem análises complexas de forma intuitiva e rápida.

Cada página deste projeto é dedicada a um aspecto específico do SIADE, como a análise de dados, visualização de resultados e geração de relatórios.
